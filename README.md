# f1-analysis
Flask web application for Formula 1 race data analysis and visualization

Choose a season, race, session type, and drivers
Compare lap times and cumulative gaps between drivers
Explore tyre strategies and stint lengths
See team pace distributions and lap time histograms
View race replay telemetry data (speed/position over the lap)
